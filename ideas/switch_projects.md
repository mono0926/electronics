# 6個のスイッチを活用した電子工作アイディア

3Dプリンターでおもちゃを作る際に余った6個のキースイッチを活用するためのアイディア集です。

## 1. 共通で必要なもの（持っていない場合）

電子工作の基礎セットです。すでにお持ちの場合は不要です。

| 品名                 | 概要                                                            | Amazon概算価格 |
| :------------------- | :-------------------------------------------------------------- | :------------- |
| **配線材**           | ジャンプワイヤ (オス-オス、オス-メス)                           | ¥500〜¥800     |
| **はんだセット**     | はんだごて、はんだ、こて台                                      | ¥2,000〜¥3,000 |
| **ユニバーサル基板** | スイッチを固定・配線する場合に便利 (3Dプリントで直配線なら不要) | ¥500           |

---

## 2. プログラマブル・マクロパッド (実用性No.1)

PCにつないでショートカットキーとして使います。
Zoom/Meetのミュート、コピペ専用キー、音楽再生コントロールなどに便利です。

| 品名             | 必須 | 概要                                                 | Amazon概算価格          |
| :--------------- | :--- | :--------------------------------------------------- | :---------------------- |
| **Pro Micro**    | ◎    | ATmega32U4搭載の互換機 (3個セットがお得なことが多い) | ¥1,500〜¥2,000 (1〜3個) |
| **キーキャップ** | 〇   | 10個セット (無刻印やクリア)                          | ¥800〜¥1,000            |
| **ダイオード**   | △    | 1N4148 (6個なら直結で不要な場合もあるが、あると確実) | ¥300〜¥500 (50-100本入) |
| **合計**         |      |                                                      | **約 ¥2,600 〜 ¥3,500** |

**ポイント:** Pro Microは「USB HID」としてPCにキーボードとして認識させることができるため、この用途に必須です。

---

光った順序を覚えるサイモンゲームや、光った場所を押すモグラ叩きゲームが作れます。

| 品名                  | 必須 | 概要                                          | Amazon概算価格          |
| :-------------------- | :--- | :-------------------------------------------- | :---------------------- |
| **Raspberry Pi Pico** | ◎    | Pythonで書きやすく安価。Pro Microでも可       | ¥800〜¥1,200            |
| **LEDセット**         | ◎    | 3mm/5mmの各色セット                           | ¥800〜¥1,200            |
| **抵抗セット**        | ◎    | LED電流制限用 (220Ω〜1kΩ程度が入っているもの) | ¥800〜¥1,000            |
| **電子ブザー**        | ◎    | アクティブブザー (電圧かけるだけで鳴る)       | ¥500〜¥800 (複数個入)   |
| **電池ボックス**      | ◎    | 単3×2本 または 3本 (スイッチ付きが便利)       | ¥500〜¥800              |
| **合計**              |      |                                               | **約 ¥3,400 〜 ¥5,000** |

**ポイント:** ラズパイPicoは安くて高性能なので、今後の電子工作にも使い回しが効きます。
