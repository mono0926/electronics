# フェーズ 2.4: ピアノへの取り付けと最終接続 🎹✨

いよいよ最後のステップです！これまでに作成したLEDバー、コントローラー、ユニットをピアノに設置し、魔法が起きる瞬間（通電テスト）の準備をします。

---

## 🛠 準備するもの

- [ ] 加工済みの **LEDアルミバー**
- [ ] 組み立て済みの **コントローラーユニット** (ラズパイ+LCD)
- [ ] **電源供給基板** (ヒューズ+コンデンサ)
- [ ] **ケーブルタイ / ケーブルクリップ**
- [ ] **養生テープ / 両面テープ**
- [ ] **USBケーブル** (ピアノ 〜 Raspberry Pi 接続用)

---

## 1. LEDバーの設置

ピアノの鍵盤のすぐ上にLEDバーを配置します。

1. **位置合わせ**: LEDの光がちょうど鍵盤を照らすように、バーの角度を調整します。
2. **固定**:
   - ピアノを傷つけないよう、アルミバーの裏面にフェルトやクッション材を貼るのがおすすめです。
   - 養生テープや弱粘着の両面テープで、演奏中に動かない程度に固定します。
3. **配線の取り回し**: コネクタがピアノの端（通常は左側）に来るように配置します。

---

## 2. ユニットの配置と配線整理

コントローラーなどは見えない場所に隠すか、操作しやすい位置にまとめます。

1. **コントローラーの設置**: Raspberry Pi はピアノの背面や棚の隅など、熱がこもらない場所に置きます。
2. **大電流ラインの接続**:
   - ヒューズ・コンデンサ回路を介して、ACアダプタ、LED、ラズパイを全て接続します。
   - コネクタ（JST-SM）をカチッと音がするまで差し込みます。
3. **ケーブルマネジメント**:
   - ピアノの演奏の邪魔にならないよう、ケーブルタイでまとめて固定します。
   - 「足を引っ掛ける」のを防ぐため、床を這う配線は最小限にします。

---

## 3. 初回通電テスト（プレフライトチェック）

いきなり電源をオンにする前に、最後の一分間の指差し確認です。

- [ ] **ショートチェック**: +5V と GND が触れている箇所はないか？
- [ ] **極性チェック**: ACアダプタのプラスマイナス、コンデンサの向きは正しいか？
- [ ] **MIDI接続**: 電子ピアノのUSB-B端子と Raspberry Pi のUSBポートを専用アダプタで繋いでいるか？

> [!CAUTION]
> 指差し確認が終わったら、**祈りとともに**ACアダプタをコンセントに差し込んでください。

---

## 4. ステータスディスプレイの確認

電源が入ると、1.44" LCD HAT に情報が表示され始めます。

- **booting...**: 起動中
- **IP Address**: このアドレスを使って設定画面にアクセスします。
- **LED OK**: 制御プログラムが正しくLEDを認識したサインです。

---

## 🏁 最終完了チェックリスト

- [ ] LEDバーが美しく水平に配置されている。
- [ ] 演奏中に配線が手に触れないように整理されている。
- [ ] 電源を入れて、ラズパイが正常に起動した。

おめでとうございます！これでハードウェアの組み立ては全て完了です。🎉
