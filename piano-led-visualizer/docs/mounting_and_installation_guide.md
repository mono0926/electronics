# 設置・固定ガイド 🛠️

プログラムのセットアップが終わったら、次は LED テープをピアノに物理的に取り付けるフェーズです。美しく、かつ鍵盤と位置がぴったり合うように設置するためのコツをまとめました。

---

## 🎹 設置位置の決定

LED テープは「鍵盤の奥側（ヒンジ付近）」に設置するのが一般的です。

1. **向きの確認**: LED テープの信号入力側（矢印 `▶` がある方）が Raspberry Pi 側にくるように配置します。
2. **位置合わせ**: ピアノの「中央のド (C4)」の真上に LED がくるように微調整します。
   - ※後でソフトウェア側でずらすことも可能ですが、物理的に合っている方が綺麗です。

---

## 📦 固定の選択肢

見た目と手間のバランスで、いくつかのアプローチがあります。

### 1. アルミニウム LED プロファイル（推奨）

最もプロフェッショナルな仕上がりになります。

- **メリット**: LED の粒感が消え（ディフューザー使用時）、光が均一に広がります。放熱効果も高いです。
- **方法**: プロファイルに LED テープを貼り、それをピアノの上に置くか、弱粘着の両面テープで固定します。

### 2. シリカゲル・チューブ

柔らかい素材で、ピアノの湾曲にも対応しやすいです。

- **メリット**: 柔らかな光になり、ピアノを傷つけにくいです。
- **方法**: チューブの中に LED テープを通し、ピアノの段差に置きます。

### 3. 直接貼り付け

最もシンプルですが、剥がす際に工夫が必要です。

- **注意**: ピアノの塗装を傷めないよう、**養生テープ（マスキングテープ）**をまずピアノに貼り、その上から LED テープの粘着面を貼るのが「賢いやり方」です。

---

## 🔌 配線の取り回しと保護

### ケースへの収容

Raspberry Pi Zero は非常に小さいため、3D プリンターで専用ケースを作成して、ピアノの端や裏側に隠すのがおすすめです。

- コンポジット（複合）ケーブルを使うと、電源と MIDI をまとめてスッキリさせられます。

### レベルシフタ (74AHCT125) の設置

以前お話しした「信号を 3.3V から 5V に持ち上げるチップ」を使用する場合：

- Raspberry Pi と LED テープのなるべく近く（数 cm 以内）にレベルシフタを配置してください。
- 配線が長くなるとノイズを拾いやすくなるため、可能であれば小さな基板にまとめてケース内に収めるのが理想です。
