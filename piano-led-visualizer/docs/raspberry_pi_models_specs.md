# 歴代 Raspberry Pi スペック比較表

Raspberry Pi（ラズパイ）の歴代主要モデルのスペックを比較した表です。
プロジェクトの要件や用途に合わせて適切なモデルを選択する際の参考にしてください。

## フラッグシップモデル (Model B シリーズ)

標準的なサイズで、充実したインターフェース（複数のUSB、有線LANなど）を備えたメインストリームのシリーズです。

| モデル名                    | 発売年 | CPUコア数 / SoC                | クロック周波数 | 基本メモリ (RAM)             | 主な進化・特徴                                                                                       |
| :-------------------------- | :----- | :----------------------------- | :------------- | :--------------------------- | :--------------------------------------------------------------------------------------------------- |
| **Raspberry Pi 1 Model B**  | 2012年 | 1コア (ARM11 / BCM2835)        | 700 MHz        | 512 MB                       | 初代機。2つのUSBポートと100Mbpsの有線LANを搭載。                                                     |
| **Raspberry Pi 2 Model B**  | 2015年 | 4コア (Cortex-A7 / BCM2836)    | 900 MHz        | 1 GB                         | 初のクアッドコア化で性能が大幅向上。メモリも1GBに倍増。                                              |
| **Raspberry Pi 3 Model B**  | 2016年 | 4コア (Cortex-A53 / BCM2837)   | 1.2 GHz        | 1 GB                         | 初の64bit対応。**Wi-FiとBluetoothを標準搭載**しIoTの定番に。                                         |
| **Raspberry Pi 3 Model B+** | 2018年 | 4コア (Cortex-A53 / BCM2837B0) | 1.4 GHz        | 1 GB                         | 3Bの改良版。クロックアップ、Wi-Fiの5GHz帯対応、Gigabit Ethernet(内部USB2.0接続)。                    |
| **Raspberry Pi 4 Model B**  | 2019年 | 4コア (Cortex-A72 / BCM2711)   | 1.5 GHz        | 1GB / 2GB / 4GB / 8GB        | アーキテクチャを一新し真のPCライクに。USB 3.0、真のGigabit LAN、デュアル4Kモニター対応。             |
| **Raspberry Pi 5**          | 2023年 | 4コア (Cortex-A76 / BCM2712)   | 2.4 GHz        | 4GB / 8GB (将来的には16GBも) | Pi 4の約2〜3倍の圧倒的性能。独自チップ「RP1」搭載でI/Oが高速化し、PCIe接続も可能に。電源ボタン追加。 |

## 小型・省電力モデル (Zero / Model A シリーズ)

フットプリントを極限まで削り、組み込み用途や低消費電力を重視したシリーズです。

| モデル名                       | 発売年 | CPUコア数 / SoC                | クロック周波数 | 基本メモリ (RAM) | 主な進化・特徴                                                                             |
| :----------------------------- | :----- | :----------------------------- | :------------- | :--------------- | :----------------------------------------------------------------------------------------- |
| **Raspberry Pi 3 Model A+**    | 2018年 | 4コア (Cortex-A53 / BCM2837B0) | 1.4 GHz        | 512 MB           | 3B+の性能を持ちつつ、基板をハーフサイズにし、LANとUSBを削減したモデル。                    |
| **Raspberry Pi Zero**          | 2015年 | 1コア (ARM11 / BCM2835)        | 1 GHz          | 512 MB           | 超小型・超安価モデル。ネットワーク機能はなし。端子はMini HDMIとMicro USB。                 |
| **Raspberry Pi Zero W / WH**   | 2017年 | 1コア (ARM11 / BCM2835)        | 1 GHz          | 512 MB           | 初代Zeroに**Wi-FiとBluetoothを追加**したもの。WHはGPIOピンヘッダ実装済み版。               |
| **Raspberry Pi Zero 2 W / WH** | 2021年 | 4コア (Cortex-A53 / RP3A0)     | 1 GHz          | 512 MB           | **Pi 3相当のクアッドコア**になり大幅に性能向上（初代Zeroの数倍）。本プロジェクトの推奨機。 |

## マイコンボード (Pico シリーズ)

Linux OSを搭載するパソコンではなく、「Arduino」のように単一のプログラムを高速・確実に実行するのに特化した超低価格マイコン基板です。

| モデル名                     | 発売年 | CPUコア数 / SoC             | クロック周波数 | 基本メモリ (SRAM) | 主な進化・特徴                                                                  |
| :--------------------------- | :----- | :-------------------------- | :------------- | :---------------- | :------------------------------------------------------------------------------ |
| **Raspberry Pi Pico**        | 2021年 | 2コア (Cortex-M0+ / RP2040) | 133 MHz        | 264 KB            | ラズパイ財団初の独自開発マイコン。500円程度で買える超低価格と扱いやすさが人気。 |
| **Raspberry Pi Pico W / WH** | 2022年 | 2コア (Cortex-M0+ / RP2040) | 133 MHz        | 264 KB            | PicoにWi-Fiモジュール（Infineon CYW43439）を追加したIoT向けモデル。             |
| **Raspberry Pi Pico 2**      | 2024年 | 2コア (Cortex-M33 / RP2350) | 150 MHz        | 520 KB            | 新チップ搭載でより強力に。セキュリティ機能も強化。                              |

## パソコン一体型 (Keyboard シリーズ)

キーボードの中に基板が全て収まっており、モニターと電源を繋ぐだけですぐにPCとして使えるモデルです。

| モデル名             | 発売年 | ベース機          | クロック周波数 | メモリ (RAM) | 主な進化・特徴                                                           |
| :------------------- | :----- | :---------------- | :------------- | :----------- | :----------------------------------------------------------------------- |
| **Raspberry Pi 400** | 2020年 | Pi 4 Model B 相当 | 1.8 GHz        | 4 GB         | 放熱性が良く、通常のPi 4（1.5GHz）よりもさらにクロックアップされている。 |
| **Raspberry Pi 500** | 2024年 | Pi 5 相当         | 2.4 GHz        | 8 GB         | Pi 5ベースの最新一体型モデル。                                           |

---

> [!TIP]
> **性能の目安**
>
> - **Zero 2 W**: 少し前の安いスマホ程度。単一の軽いタスク（Webサーバー、LED制御、センサー読み取り）には優秀で省電力。
> - **Pi 4**: エントリークラスのChromebook程度。ブラウザで複数タブを開いたり、簡単な動画再生ができる。
> - **Pi 5**: 少し前のMacBook Airや一般的なノートPCに迫る性能。AI処理や重いデスクトップ用途もサクサク動く。
