# Pinecil V2 使い倒しガイド

## 1. 初回セットアップ

### 電源の選び方

| 電源タイプ               | 出力         | 加熱速度                  | 評価       |
| :----------------------- | :----------- | :------------------------ | :--------- |
| USB-C PD 65W (20V/3.25A) | 65W          | 約6〜12秒で作業温度に到達 | ◎ おすすめ |
| USB-C PD 140W EPR (28V)  | 最大88W      | 最速                      | ◎ 最高性能 |
| USB-C PD 45W (20V)       | 45W          | やや遅い                  | 〇 使える  |
| QC3.0スマホ充電器        | 12V/1.5A程度 | 非常に遅い＆不安定        | ✕ 非推奨   |

> ⚠️ **QC（Quick Charge）スマホ充電器はNG。** アンペアが足りずUndervoltage警告が出たり、最悪熱暴走の原因になります。

### ⚠️ 電源に関する重要な注意事項

- **「DC Low」と表示される場合**: 電圧が足りていません。PCのUSBポートなどでは加熱できません。
- **絶対にやってはいけないこと**: USB-CとDCジャックの**2本差しは厳禁**です。本体や電源が破損します。

### こて先の取り付け

Pinecil v2は、本体とチップが分かれた状態で届きます。

1. **接点の清掃（重要！）**:
   - チップの白い部分にある2つの銀色の接点を、アルコール（IPA）や乾いた布で拭きます。工場出荷時の微量な油分が残っていると、温度表示が不安定になることがあります。
2. **装着手順**:
   - 本体先端にある小さなネジを少し緩めます。
   - チップを奥までしっかり差し込みます。チップの根元にある太いプラスチックのリングが本体に当たるまで押し込んでください。
   - 緩めたネジを軽く締めます。**締めすぎに注意**してください（コテ先が抜けなければ十分です）。

- こて先は**電源を入れる前に**取り付けること（起動時に自動検出される。ホットスワップ非推奨）
- Pinecil V2は **TS100/TS101互換チップ** と **Pine64 STシリーズ（ショートチップ）** の両方に対応
- 標準付属は **B2型**（汎用的な円錐型）

### おすすめこて先の使い分け

| 型          | 形状                 | 用途                               |
| :---------- | :------------------- | :--------------------------------- |
| **B** (B2)  | 円錐                 | 汎用。最初はこれ1本で十分          |
| **I**       | 極細円錐             | SMD部品、精密作業                  |
| **K**       | ナイフ型             | ドラッグはんだ（複数ピンを一気に） |
| **D** (D24) | マイナスドライバー型 | 大きめの部品、広い面の加熱         |

## 2. IronOS の設定

### 温度設定の目安

| はんだの種類                            | 推奨温度      |
| :-------------------------------------- | :------------ |
| **有鉛はんだ** (Sn60/Pb40) ← 今回購入分 | **300〜320℃** |
| 鉛フリーはんだ                          | 350〜365℃     |

> 💡 **コツ:** 必要最低限の温度で作業するのがこて先長持ちの秘訣。300℃から始めて、はんだが溶けにくければ5℃ずつ上げていく。

### 覚えておくと便利な操作

| 操作               | やり方                                                                         |
| :----------------- | :----------------------------------------------------------------------------- |
| はんだモードに入る | **先端側ボタン (+)** を押す                                                    |
| 設定メニューを開く | **後端側ボタン (-)** を押す                                                    |
| ブーストモード     | はんだモード中に **先端側ボタン長押し** → 一時的に高温化（大きい接合部に便利） |
| 温度の微調整       | はんだモード中に **+/-ボタン** で1℃単位調整                                    |
| 詳細表示           | Settings → UI → Detailed idle screen → ON                                      |

> 💡 **チップ側が `[+]` 、電源端子側が `[-]` ボタンです。**

### おすすめ初期設定

1. **温度:** 300〜320℃（有鉛はんだ用）
2. **スリープ温度:** 150℃（デフォルトのままでOK）
3. **スリープ時間:** 5分（デフォルト）
   - **オートスリープ & モーション喚起**: しばらく置くと自動で温度が下がり、持ち上げると加速度センサーが検知して一瞬で設定温度に戻ります。
4. **Max Power:** 使用する電源のW数に合わせる（65W充電器なら65Wに）
5. **温度単位:** ℃

## 3. ファームウェア更新 (IronOS)

IronOSはオープンソースで頻繁に改善されているので、届いたらまず更新推奨。

### 手順

1. [IronOS Releases](https://github.com/Ralim/IronOS/releases) から最新の安定版をダウンロード
   - Pinecil V2用は `Pinecilv2_JA.bin`（日本語）
2. Pinecilの電源を外す
3. **[-]ボタンを長押ししながら** USB-CケーブルでPCに接続
4. 画面が**真っ暗のまま**であればFlashモードに入れている（画面がつく場合はやり直し）
5. フラッシュツールで書き込み:
   - **macOS:** [Blisp (CLI)](https://github.com/pine64/blisp/releases) を使用
   - **GUI派:** [PineFlash](https://github.com/pine64/PineFlash) （Windows/Linux/macOS）

> ⚠️ ファームウェア更新時は**USB-Cポートのみ使用**すること。DC バレルジャックから給電しながらの更新はPCとPinecil両方壊れる危険あり。

## 4. こて先のメンテナンス

### 長持ちさせるコツ

- **作業前:** こて先にはんだを薄く乗せる（ティンニング）。酸化防止になる
- **作業中:** こて先が汚れたら**真鍮ワイヤーボール**で軽くこする（濡れスポンジより先端温度が下がりにくい）
- **作業後:** こて先にはんだを多めに盛ってから電源OFF（次回使用時の酸化を防ぐ）
- **保管時:** 長期間使わない場合はこて先を外して保管

### やってはいけないこと

- ❌ 必要以上の高温で放置（酸化が急速に進む）
- ❌ ヤスリやサンドペーパーでこて先を削る（めっきが剥がれる）
- ❌ こて先にはんだを乗せずに加熱する

---

## 5. 安全のポイント

- 使わない時は必ず**電源を抜く**
- HOZAN H-11のフタ付きこて台は、Pinecilのコンパクトさと相性◎
- 作業場所には**耐熱マット**を敷くと安心（シリコンマットがおすすめ）
- お子さんがいる環境では、**はんだの煙を吸わないよう換気**に注意（有鉛はんだは特に）
- 鉛入りはんだ作業後は**必ず手を洗う**
